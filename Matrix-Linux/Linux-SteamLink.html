How To Remotely Play Games.

There are multiple ways to play games remotely like Parsec,Moonlight,Sunshine and SteamLink.

We are going to focus on SteamLink as it is the easiest and works on almost every device.

Download And Install SteamLink

First Download Steamlink application on the system where you want to stream to (slower computer) from here.

  • Arch:

    yay -S steamlink

  • Flatpak/Others:

    flatpak install flathub com.valvesoftware.SteamLink

Troubleshooting

  • Computers cannot detect one an other:

    1. Make sure you downloaded Steamlink to both computers.

    2. Make sure both of them are running steam when searching for one another.

  • Controller Causes Double Input

    • This happens because steam makes a virtual controller so that we can map our controller's input button to anything we want. This makes "steamlink" think that there are 2 controllers when in fact there is only one of them. We just have to remove one of them.

      Windows

      1. Go to controller panel and disable one of the controller.

      2. Or use HidHide.

      3. Or follow DS4-For-Windows guide.

      GNU/Linux

      1. Follow this guide to disable a controller.

      2. Or follow Arch Wiki.

        3. Or follow my quick and dirty methord of first downloading AntiMicroX/AUR

        1. Open AntiMicroX.

        2. Map 1 button.

        3. Open Steam and SteamLink.

        4. Steam will see only one controller.
